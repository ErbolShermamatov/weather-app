# Weather App

Приложение для просмотра погоды в реальном времени.
Использует OpenWeatherMap API для получения данных.

[Открыть сайт](https://erbolshermamatov.github.io/weather-app/)

## Технологии
* **HTML5 / CSS3** (Flexbox, Адаптивная верстка)
* **JavaScript (ES6+)**
* **Fetch API** (Асинхронные запросы)
* **Async / Await**

## Функционал
* Поиск погоды по названию города.
* Отображение температуры, влажности и ветра.
* Динамическая смена иконки погоды (Дождь, Солнце, Облака).
* Обработка ошибок (если город не найден).
